Dot files for Linux (and maybe some \*nix)
=========================================

Dependencies:

* Font Awesome 4.0+
* Google Fonts
* GNU Stow
* xorg-xbacklight (for brightness controls)
* compton

For bash:

* powerline

For i3blocks:

* acpi
* amixer

For vim:

* vundle

Setup:
`stow <application_name>`
